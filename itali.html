Resumen de la orden

Inventario, ventas y compras — todos los SKUs

Organizado por las mismas familias del catalogo de italianaferramenta.it. Toca una familia para abrirla. El punto verde marca familias con lineas en orden. Escribe una cantidad en Ordenar para agregar el SKU a la orden. STD/12 con ⚠ = tasa inflada por venta historica (SKU viejo).

Base de datos de productos

Fuente IF = descripcion oficial de la guia o cotizacion de Italiana Ferramenta. Fuente EPICOR = traduccion automatica pendiente de revision (ambar). Fotos: enlace directo al sitio de IF con respaldo embebido para codigos custom.
